Class: Profile

Profile()

This is a React component that renders a dropdown menu for user profiles. It shows different options for logged-in users and guests. Logged-in users can access their favorite list and log out, while guests are offered a login/registration option. The dropdown toggle button displays the user's avatar or a default guest image if the user is not logged in.
