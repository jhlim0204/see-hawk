Class: state

SearchPage#state(isLoading, oriSearchResult, searchResult, star, region)

new state(isLoading, oriSearchResult, searchResult, star, region)

Represents the state of a component that contains properties related to search results, loading status, star count, and region filtering.
Parameters:
Name Type Description
isLoading boolean A boolean value indicating whether the search results are currently being loaded.
oriSearchResult Array An array of original search results.
searchResult Array An array of filtered search results.
star number A numeric value representing the number of stars for a given search result.
region Object An object containing boolean properties representing the region filter for search results.
Properties
Name Type Description
W boolean A boolean value indicating whether search results are filtered by the western region.
E boolean A boolean value indicating whether search results are filtered by the eastern region.
N boolean A boolean value indicating whether search results are filtered by the northern region.
NE boolean A boolean value indicating whether search results are filtered by the northeastern region.
C boolean A boolean value indicating whether search results are filtered by the central region.
